## 圖片規格建議

| 類型 | 建議尺寸 | 格式 | 壓縮目標 |
|------|----------|------|----------|
| Hero 背景 | 1920x1080px | JPG/WebP | < 200KB |
| Page Header | 1920x600px | JPG/WebP | < 150KB |
| 卡片縮圖 | 400x300px | JPG/WebP | < 50KB |
| 照片集 | 600x600px | JPG/WebP | < 80KB |
| Logo | 適當大小 | PNG (透明背景) | < 30KB |
| 講者照片 | 300x300px | JPG | < 30KB |

## 圖片優化工具

1. **TinyPNG** - https://tinypng.com/
2. **Squoosh** - https://squoosh.app/
3. **ImageOptim** (Mac) - https://imageoptim.com/

## 注意事項

1. 使用 WebP 格式可減少 25-35% 檔案大小
2. 確保所有圖片有適當的 alt 文字
3. 使用 lazy loading 提升頁面載入速度
4. 考慮提供 @2x 版本支援 Retina 顯示器
5. 所有素材使用前請確認版權授權
