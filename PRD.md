# 花蓮隊計畫網站 PRD（產品需求文件）

## 一、專案概述

### 1.1 專案名稱
花蓮隊計畫官方網站改版

### 1.2 專案目標
建置一個專業、現代化的政府計畫宣傳網站，以強化施政行銷宣傳，讓民眾快速了解計畫內容與成果。

### 1.3 參考網站
- 舊網站：https://teamhualien.com/zh/
- 風格參考：https://reap.mit.edu/ （MIT REAP）

### 1.4 目標受眾
- 一般民眾
- 產業界人士
- 國際合作夥伴
- 政府相關單位
- 媒體記者

---

## 二、設計規範

### 2.1 設計風格
專業、簡潔、企業學術風格（Corporate Academic Style）

### 2.2 色彩規範
| 用途 | 色彩名稱 | 色碼 |
|------|----------|------|
| 主色 | Cardinal Red | #A31F34 |
| 輔色 | 深灰色 | #333333 |
| 輔色 | 銀色 | #C0C0C0 |
| 背景 | 純白 | #FFFFFF |
| 文字 | 深灰 | #4A4A4A |

### 2.3 字型規範
- 中文：Noto Sans TC / 思源黑體
- 英文：Helvetica Neue / Roboto
- 標題：Bold / Semi-Bold
- 內文：Regular, 16px 以上

### 2.4 響應式斷點
| 裝置 | 斷點範圍 |
|------|----------|
| 桌機 | ≥ 1200px |
| 平板 | 768px - 1199px |
| 手機 | < 768px |

### 2.5 視覺設計要點
- 大量留白（Whitespace）
- 簡潔的導覽列
- 扁平化設計元素
- 專業的 Banner 圖片搭配淡色疊加層
- 結構化網格系統
- 卡片式 UI 設計
- 資料視覺化呈現

## 三、功能需求

### 3.1 前台功能

#### 3.1.1 首頁（Home）
| 功能區塊 | 說明 |
|----------|------|
| Hero Banner | 大型專業橫幅，含主視覺、標語、影片播放按鈕 |
| 最新消息 | 顯示最新 3-5 則消息，卡片式呈現，含日期與「閱讀更多」按鈕 |
| 熱門訊息快捷鍵 | 重要訊息或功能的快速連結區塊 |
| 關鍵字搜尋 | 全站搜尋功能，支援中英文搜尋 |
| 服務窗口 | Footer 區域顯示地址、電話、Email 等基本資訊 |
| 互動地圖 | 扁平化世界/台灣地圖，標示計畫據點（選配） |

#### 3.1.2 計畫內容（About the Project）
| 頁面 | 說明 |
|------|------|
| 計畫背景 | 計畫緣起與重要性說明 |
| 計畫沿革 | 時間軸呈現計畫發展歷程 |
| 計畫目標 | 條列式呈現計畫核心目標 |
| 工作坊紀實 | 圖文並茂呈現各場次工作坊記錄 |

#### 3.1.3 倡議計畫（Initiatives）
| 子項目 | 說明 |
|--------|------|
| 大健康產業園區 | 專屬介紹頁面，含概念說明、規劃內容、預期效益 |
| 中草藥示範基地 | 專屬介紹頁面，含基地規劃、作物種類、發展願景 |
| 共享廚房 | 專屬介紹頁面，含營運模式、服務對象、場地介紹 |
| 綠色觀光產業療癒圖譜 | 專屬介紹頁面，含路線規劃、景點介紹、體驗內容 |

#### 3.1.4 國際論壇（International Forum）
| 功能 | 說明 |
|------|------|
| 論壇議程 | 時間表式呈現，可依日期/主題篩選 |
| 講者介紹 | 卡片式呈現，含照片、姓名、職稱、簡介 |
| 報名連結 | 外部連結或內嵌報名表單 |

#### 3.1.5 計畫成果（Results）
| 功能 | 說明 |
|------|------|
| 成果報告 | 可下載的 PDF 報告列表 |
| 媒體曝光 | 媒體報導連結彙整，含來源與日期 |
| 照片集 | 相簿式瀏覽，支援 Lightbox 放大檢視 |
| 影片集 | 影片嵌入播放（YouTube/Vimeo） |

#### 3.1.6 合作單位（Partners）
| 功能 | 說明 |
|------|------|
| LOGO 牆 | 網格式排列各合作單位 LOGO |
| 連結功能 | 點擊 LOGO 可連結至該單位官網（選配） |
| 分類顯示 | 依單位類型分類呈現（選配） |

#### 3.1.7 聯絡我們（Contact Us）
| 功能 | 說明 |
|------|------|
| 聯絡資訊 | 顯示電話、Email、地址 |
| 聯絡表單 | 線上留言表單（姓名、Email、主旨、內容） |
| Google Map | 嵌入地圖顯示辦公室位置 |

#### 3.1.8 共用功能
| 功能 | 說明 |
|------|------|
| 中英文切換 | Header 提供語言切換按鈕 |
| RWD 響應式 | 自動適應桌機、平板、手機 |
| 麵包屑導覽 | 顯示當前頁面層級位置 |
| 回到頂端 | 浮動按鈕，點擊回到頁面頂端 |
| 社群分享 | 各頁面可分享至 Facebook、LINE 等 |

## 四、技術規格

### 4.1 建議技術架構
| 項目 | 建議技術 |
|------|----------|
| 前端框架 | Next.js / React |
| 樣式框架 | Tailwind CSS |
| 後端框架 | Node.js / Express 或 PHP / Laravel |
| 資料庫 | PostgreSQL / MySQL |
| CMS | Strapi / 自建 Admin Panel |
| 主機 | AWS / GCP / Azure |

### 4.2 效能要求
| 項目 | 標準 |
|------|------|
| 首頁載入時間 | ≤ 3 秒 |
| 頁面切換時間 | ≤ 1 秒 |
| 圖片壓縮 | WebP 格式優先 |
| 快取策略 | 靜態資源 CDN 加速 |

### 4.3 SEO 優化
- 語意化 HTML 結構
- Meta Tags 完整設定
- Open Graph 標籤
- Sitemap.xml 自動產生
- robots.txt 設定
- 結構化資料（Schema.org）
